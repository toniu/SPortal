# Tests

Unit tests for the core logic in each web part, written with Jest and React Testing Library.

| Test file | Covers |
|-----------|--------|
| `DashboardApp.test.tsx` | Dashboard component rendering and event retrieval |
| `GroupManagement.test.tsx` | Group CRUD operations and membership actions |
| `PollManagement.test.tsx` | Poll voting, analytics and chart configuration |
| `Profile.test.tsx` | Profile display and user-suggestion logic |

## Running tests

```bash
npm test
```
