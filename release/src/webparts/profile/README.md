# Profile Web Part

Displays the logged-in user's SharePoint profile — name, job title, department and group memberships — and suggests other people in the tenant to connect with.

> **Built with:** SPFx · React · PnPjs · Office UI Fabric React

---

## Features

- **Profile Card** — Shows the current user's name, title, department, and the groups they own or belong to
- **People You Might Know** — Surfaces a randomised selection of other users in the tenant for discovery
